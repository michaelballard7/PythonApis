

URLLib Notes: 

- URLLib on suppors a support of HTTP by default

- Doesnt automatically decode returned data. 

- Common features such as cookies or authentication, require more modules.

- Difficult to implement advanced features, such as timeouts

- Proccessing returned data such as JSON, is cumbersome


Requests Notes:

- Simpler API for working with HTTP operations

- Each HTTP operation maps to a method name in Requests

- Makes working with parameters,headers, and cookies easier

- Automatically decodes returned content

- Automatticaly parses JSON content when detected

- Handles redirects, timeouts and errors 

- Advanced features like authentication and sessions
